# Ethereum Node Deployment (Bash)

Production-ready bash scripts for deploying and managing Ethereum nodes.

## Features

- ✅ Multi-client support (Geth, Besu)
- ✅ Network selection (Mainnet, Sepolia, etc.)
- ✅ Docker-based deployment
- ✅ Health monitoring
- ✅ Log management

## Usage

```bash
# Deploy node
./deploy-node.sh deploy

# Check status
./deploy-node.sh status

# View logs
./deploy-node.sh logs

# Health check
./deploy-node.sh health
```
