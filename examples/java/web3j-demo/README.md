# Web3j Ethereum Demo (Java)

Enterprise-grade Ethereum integration using Web3j for Java applications.

## Features

- ✅ Account creation and management
- ✅ Balance queries
- ✅ Transaction monitoring
- ✅ Message signing
- ✅ Address validation
- ✅ Gas price queries
- ✅ Block information

## Setup

```bash
mvn clean install
mvn exec:java
```

## Why Java for Web3?

- **Enterprise Ready**: Mature ecosystem
- **Spring Integration**: Easy backend integration
- **Type Safety**: Strong typing
- **Performance**: JVM optimization
- **Cross-platform**: Write once, run anywhere
