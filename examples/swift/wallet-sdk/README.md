# Ethereum Wallet SDK (Swift)

Native iOS/macOS wallet SDK for Ethereum integration.

## Features

- ✅ Wallet creation
- ✅ Balance queries
- ✅ Transaction sending
- ✅ Message signing
- ✅ Native Swift async/await

## Setup

```bash
swift build
swift run
```

## Why Swift for Web3?

- **iOS Native**: Perfect for mobile wallets
- **Type Safety**: Strong type system
- **Performance**: Native compiled code
- **Modern**: Async/await patterns
- **SwiftUI**: Easy UI integration
