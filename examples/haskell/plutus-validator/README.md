# Plutus Validator (Haskell)

Cardano smart contract validator written in Plutus/Haskell.

## Features

- ✅ Functional smart contract
- ✅ Type-safe validation
- ✅ Datum/Redeemer pattern
- ✅ Formal verification friendly

## Why Haskell for Blockchain?

- **Formal Verification**: Mathematical correctness
- **Type Safety**: Prevents entire classes of bugs
- **Immutability**: Pure functional programming
- **Cardano**: Native Plutus language
